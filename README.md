# 🛡️ ESP32 Connectivity Watchdog (Cão de Guarda)

**Aluno:** David Lucas  
**Turma:** DS3T  

## 📌 Sobre o Projeto

O projeto **Watchdog de Conectividade** tem como objetivo transformar o ESP32 em um "cão de guarda" responsável por monitorar continuamente o status de um servidor na internet.

Instalado em ambiente industrial (como uma fábrica), o dispositivo verifica periodicamente se o servidor está ativo e responde corretamente, fornecendo um retorno visual imediato através de LEDs.

## 🎯 Missão

Como Técnico em Análise e Desenvolvimento, a missão foi desenvolver um sistema embarcado capaz de:

- Monitorar a conexão com a internet
- Realizar verificações periódicas de um servidor
- Informar o status de forma simples e visual
